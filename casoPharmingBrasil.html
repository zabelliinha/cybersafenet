CASO DE PHARMING NO BRASIL, 2015

Conhecido como o "Ataque do DNSChanger"

O DNSChanger é um tipo de malware que altera as configurações de DNS (Domain Name System) de um dispositivo, redirecionando o tráfego da Internet para servidores DNS maliciosos controlados pelos hackers.

O que foi?

Os hackers exploraram uma vulnerabilidade em roteadores domésticos para acessar o console do administrador e alterar as configurações de DNS para um servidor DNS malicioso. Ao fazer isso, os hackers podiam controlar e direcionar o tráfego da Internet dos usuários para servidores DNS maliciosos, onde poderiam redirecioná-los para sites falsos ou infectados, realizar ataques de phishing ou roubar informações sensíveis.
O ataque foi conduzido por meio de um e-mail contendo um link que redirecionava as vítimas para um servidor que comprometia o roteador.
