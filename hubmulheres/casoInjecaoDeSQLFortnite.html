CASO DE INJEÇÃO DE SQL - FORTNITE, 2019

O que ocorreu?

Em 2019, ocorreu um caso de injeção de SQL com o popular jogo Fortnite. A Check Point Software descobriu uma vulnerabilidade que poderia permitir que hackers acessassem as contas dos jogadores, bem como seus dados pessoais. Além disso, os invasores poderiam potencialmente fazer compras dentro do jogo usando os cartões de crédito dos jogadores. Havia também o risco de os criminosos terem acesso às conversas dos jogadores, tanto online quanto da própria residência da vítima.

Resultado do ataque

Felizmente, a vulnerabilidade foi corrigida. No entanto, até o momento, não há notícias confirmadas sobre se os hackers exploraram essa vulnerabilidade antes que fosse corrigida.
