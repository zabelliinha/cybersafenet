SPYWARE

O que é?

Spyware é vagamente definido como software malicioso projetado para entrar no dispositivo do seu computador, coletar dados sobre você e encaminhá-los a terceiros sem o seu consentimento. Spyware também pode se referir a software legítimo que monitora seus dados para fins comerciais, como publicidade. No entanto, spyware malicioso é explicitamente usado para lucrar com dados roubados.

Ações do Spyware no dispositivo

- Infiltrar-se — por meio de um pacote de instalação de aplicativo, site malicioso ou anexo de arquivo.

- Monitora e captura dados — por meio de pressionamentos de teclas, capturas de tela e outros códigos de rastreamento.

- Envia dados roubados — ao autor do spyware, para serem usados diretamente ou vendidos a terceiros.

Quatro categorias principais de Spyware:

- O spyware Trojan entra nos dispositivos por meio do malware Trojan, que distribui o programa spyware.

- O adware pode monitorar você para vender dados a anunciantes ou veicular anúncios maliciosos enganosos.

- Arquivos cookies de rastreamento podem ser implantados por um site para segui-lo pela Internet.

- Os monitores do sistema rastreiam qualquer atividade em um computador, capturando dados confidenciais, como pressionamentos de teclas, sites visitados, e-mails e muito mais. Keyloggers normalmente se enquadram neste grupo.
