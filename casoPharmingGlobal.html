CASO DE PHARMING GLOBAL 2007

O que foi?

Em 2007, mais de 50 instituições financeiras em todo o mundo foram alvo de um sofisticado ataque de pharming. Os hackers criaram sites falsos para cada alvo e injetaram código malicioso, forçando os computadores das vítimas a baixar um malware. Este malware, por sua vez, baixou arquivos adicionais de um servidor russo. Quando os usuários tentavam acessar os sites reais das instituições financeiras, eram redirecionados para os sites falsos, onde suas credenciais eram capturadas antes de serem redirecionados para os sites reais. Este ataque afetou milhões de pessoas em várias partes do mundo, explorando uma vulnerabilidade nos softwares para atrair clientes de bancos para os sites falsos.
